# Fastapi - project

Prosta aplikacja w fastapi do obsługi bazy danych z filmami.
Zawiera też kilka innych endpointow z zajęć: operacje arytmetyczne i
pobieranie danych geolokalizacyjnych.

## Technologie
- Python
- FastApi

## Wymagania

requirements.txt

## Instalacja i uruchomienie

1. Klonowanie repozytorium:

git clone https://github.com/rayoush1/fastapi-project4.git

2. Instalacja zależności:

pip install -r requirements.txt

3. Uruchomienie

W terminalu przejdź do folderu z repozytorium i wywołaj

fastapi dev main.py

Strona powinna działać na http://127.0.0.1:8000/

## Endpointy

Listę endpointów można zobaczyć na 

http://127.0.0.1:8000/docs
lub
http://127.0.0.1:8000/redoc
